There are two ways to save, Private or Pretty.

Method Description Example Link
Pretty Pretty is easy to read, you can print it out and it will make some kind of sense. Obviously that means nosy parkers trying to find out what you're getting them could read this though - with great power comes great responsibility!
Private Private lists are not readable by humans. They're encoded so that comuters can read them, but if someone got their hands on your list, they'd need to do some work on it before they could read it in any way that made sense
